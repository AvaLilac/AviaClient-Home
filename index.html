Avia Client adds Plugins, QuickCSS, Themes, font loading, and more on top of Stoat available as a desktop app, userscript, or browser build.

Features

🧩

Full Plugin Support

Install community plugins or write your own with JavaScript. Local plugins supported too.

🎨

Quick CSS

Modify the desktop client directly with CSS live, no reload required.

🖌️

Themes

A dedicated Themes panel so you don't have to rely on QuickCSS. Create themes right inside the UI.

🔤

Font Loader & Remover

Load custom fonts into the app, or restore the default font with one click.

⭐

Favourites System

Save and quickly access your favourite YouTube links and images.

📦

Local Plugins

Load plugins from your own machine with the new Local Plugins UI. View source code via the built-in View button.
